## :hammer: &nbsp; How to run the code

First, you need to configure your `.env` and set your keys:

```sh
cp .env.test .env
```

Then just run the following command:

```sh
yarn start
```

## :wrench: &nbsp; Tests
Tests are available by running the following command:
```sh
yarn test a
```
